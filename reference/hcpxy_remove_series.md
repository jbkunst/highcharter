# Remove series to higchartProxy element

Remove series to higchartProxy element

## Usage

``` r
hcpxy_remove_series(proxy, id = NULL, all = FALSE)
```

## Arguments

- proxy:

  A `higchartProxy` object.

- id:

  A character vector indicating the `id` (or `id`s) of the series to
  remove.

- all:

  A logical value to indicate to remove or not all series. The values is
  used only when the value is `TRUE`.
