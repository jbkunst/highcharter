# Show or hide loading text for a higchartProxy object

Show or hide loading text for a higchartProxy object

## Usage

``` r
hcpxy_loading(proxy, action = "show")
```

## Arguments

- proxy:

  A `higchartProxy` object.

- action:

  Single-element character vector indicating to `"show"` or `"hide"` the
  loading text defined in `lang` options.
