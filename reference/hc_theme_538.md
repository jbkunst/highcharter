# Theme collection for highcharts

Highcharts is very flexible so you can modify every element of the
chart. There are some exiting themes so you can apply style to charts
with few lines of code.

## Usage

``` r
hc_theme_538(...)

hc_theme_sparkline_vb(...)

hc_theme_tufte2(...)
```

## Arguments

- ...:

  A named parameters to modify the theme.

## Examples

  hc_add_theme(hc_theme_538())
