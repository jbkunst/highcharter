# Define aesthetic mappings using strings. Similar in spirit to `ggplot2::aes_string`

Define aesthetic mappings using strings. Similar in spirit to
[`ggplot2::aes_string`](https://ggplot2.tidyverse.org/reference/aes_.html)

## Usage

``` r
hcaes_string(x, y, ...)

hcaes_(x, y, ...)
```

## Arguments

- x, y, ...:

  List of name value pairs giving aesthetics to map to variables. The
  names for x and y aesthetics are typically omitted because they are so
  common; all other aesthetics must be named.
