# `hc_add_series` for numeric objects

`hc_add_series` for numeric objects

## Usage

``` r
# S3 method for class 'numeric'
hc_add_series(hc, data, ...)
```

## Arguments

- hc:

  A `highchart` `htmlwidget` object.

- data:

  A numeric object

- ...:

  Arguments defined in
  <https://api.highcharts.com/highcharts/plotOptions.series>.
