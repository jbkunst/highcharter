# Widget render function for use in Shiny

Widget render function for use in Shiny

## Usage

``` r
renderHighchart(expr, env = parent.frame(), quoted = FALSE)

renderHighchartZ(expr, env = parent.frame(), quoted = FALSE)
```

## Arguments

- expr:

  A highchart expression.

- env:

  A environment.

- quoted:

  A boolean value.
